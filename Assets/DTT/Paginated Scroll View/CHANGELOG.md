# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.3] - 2022-10-27
### Fixed
- Unclickable input field for the page in the Asset Store demo.
### Updated
- Support for the world space canvas render mode and rotated canvases.

## [2.0.2] - 2022-08-24
### Fixed
- Offset issue on start of the game.
- Paginated buttons not updating on start.

### Updated
- Updated dependencies.

## [2.0.1] - 2022-05-19
### Updated
- Updated dependencies.

### Removed
- Removed changelog from documentation.

## [2.0.0] - 2022-04-13
### Added
 - Paginated scroll now has an amount of pages changed event for future features.

### Updated
- Page Dot now has color options for pagenumbers displayed within the dots.
- Page Dots Manager now has an index offset option to alter the actual and displayed page index.
- Page Dots will no longer break after changing scenes.

### Removed
- Unusable page dot scripts have been removed.

## [1.0.2] - 2022-04-08
### Added
- Asset video demo scene.
- Sprites sued for asset video demo scene.
- Page dot buttons text now starts numbering from 1.
- Documentation update.

## [1.0.1] - 2022-03-22

### Fixed
- Spelling in editor function name
- Documentation pdf and editor Test.

## [0.0.0] - 2022-2-1
### Added
- Example scenes for horizontal and vertical scrolling.
- The paginated scroll view script works now.
